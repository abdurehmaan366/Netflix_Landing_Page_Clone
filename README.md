# Netflix Clone (HTML & CSS)

A front-end clone of the Netflix Pakistan landing page built using only HTML and CSS.

## Features

* 🎬 Netflix-inspired hero section
* 🌑 Dark theme UI
* 📈 Trending movies horizontal carousel
* ✨ Hover animations and transitions
* 📋 Interactive FAQ accordion
* 🎨 Custom curved divider with SVG
* 📱 Clean and modern layout
* 🌐 Footer and language selector

## Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Material Icons
* SVG Graphics

## What I Learned

* CSS Flexbox
* CSS Grid
* Positioning (relative & absolute)
* CSS transitions and hover effects
* Building accordions using `<details>` and `<summary>`
* Creating modern UI layouts
* Working with SVG shapes and overlays
